# React CRUD com npm
- Usa JSX, componentes, @emotion/styled, props, useEffect, useState, fetch, map, form  
Para somente iniciar:
- ```npm install```
- ```npm start```  
Comandos que usei para fazer o projeto:
- ```npm init -y```
- ```npm install esbuild```
- ```npm install react react-dom```
- ```npm run build```
- ```npm start```
